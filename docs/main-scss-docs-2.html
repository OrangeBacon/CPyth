SCSS mixins

Mixins used in CPyth's SCSS files

Size mixin

A scss mixin that can set the height and width properties at the same time.

If the width is not provided, the default is the height, making it square.

height:

The height that you want it to be.

width:

The width that you want it to be.

none

Mixins do not return values, instead set css values
